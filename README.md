# agentds-goobert-the-goblin-solution
Solutions for the [AgentDS-Bench](agentds.org) competition. Won the Manufacturing &amp; Commerce domains.

## Download official AgentDS datasets
- Instructions are provided here: [AgentDS dataset link](https://huggingface.co/datasets/lainmn/AgentDS).
- Move datasets to your working directory `agentds-goobert-the-goblin-solution/{domain}/datasets/`.

## Work on AgentDS challenges
Baseline notebooks for the Manufacturing & Commerce domains are provided in `{domain}/models/Challenge{num}_baseline.ipynb`. Try to beat our scores!
